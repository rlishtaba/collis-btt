# Collis::Btt

UL / Collis Brand EMV Test Tool automation module. See details about hardware https://www.collistesttools.com/ 
## Installation

Add this line to your application's Gemfile:

```ruby
gem 'collis-btt'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install collis-btt
